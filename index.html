RetroTube

A stripped down YouTube user interface, for those of us who prefer a simpler time in the days of the internet.

Built with React, this makes use of the YouTube API to deliver video content in a simple, clean interface.

8bit Paint

8bit Paint allows the user to paint a 16x16 grid in a retro sprite style and can even download the image when complete!

Based on my love for all things Nintendo!

github.com/GlenCarpenter/8bit_paint

React Key Logger

React Key Logger is an open source proof-of-concept to show how user behavior can be tracked and measured.

The question is, how useful is it to harvest ALL user keystrokes on your app? Is it ethical to do so? The key logger was fairly easy to create, however the privacy implications must be carefully considered prior to deployment.

github.com/GlenCarpenter/reactKeyLogger

Montessori Household

Montessori Household is a simple Wordpress eCommerce website that delivers tools to develop a Montessori learning environment in individual households.

The site was initially created as an market evaluation for the products being sold, which was then updated to a full fledged web store.

VetRelief Site Enhancement

VetRelief Site Enhancement was created per a customer request on up Upwork.com.

This is a Chrome extension that changes layout of VetRelief.com website, which is owned and maintained by a third party developer. This is accomplished by use of jQuery plugins and HTML/CSS injection.

Reddit User Search

Reddit User Search is a Chrome extension that allows you to search a Reddit User's profile and posts with ease.

github.com/GlenCarpenter/RedditUserApp

Poetry Simulator

Poetry Simulator uses the DataMuse API to find rhyming and related words to automatically generate a poem, based off of a 'seed' word provided by the user.
